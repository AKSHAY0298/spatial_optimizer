# spatial_optimizer

6G tower placement optimization tools and visualizations.

Quick start
-----------

1. Create and activate a Python virtual environment:

	python3 -m venv .venv
	source .venv/bin/activate
	# On Windows PowerShell:
	.venv\Scripts\Activate.ps1
	# On Windows Command Prompt:
	.venv\Scripts\activate.bat

2. Install dependencies (if a requirements file exists):

	pip install -r requirements.txt

3. Run the main scripts or server:

	python run.py

What is included
-----------------
- Core optimization and spatial utilities in the `core/` package.
- Example data in the `data/` folder.
- Scripts and notebooks for visualization and parameter tuning.
